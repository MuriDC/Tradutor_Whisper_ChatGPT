# Tradutor_Whisper_ChatGPT
É possível fazer a tradução de qualquer idioma para qualquer idioma, suportados pela ferramenta whish e o chat gpt. Para isso é necessário alterar somente as duas primeiras variáveis, a linguagem de entrada e saída. 

# Executando o projeto
Para capturar um audio, transformar em texto, e utilizar o chat GPT para responder, fazendo a trasnformação dessa resposta de texto para audio é necessário fazer:
Gravação de Áudio Com Python;
Reconhecimento de Fala com Whisper;
Integração com a API do ChatGPT;
Sintetizara Resposta do ChatGPT Como Voz (gTTS);
